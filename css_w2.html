jenis css

  1. inline css

    2. h2 style="color: blue; ->menjadi atribut; dimasukkan dalam tag style

  2. internal css

    3. berada dalam satu halaman html tetapi tidak di dalam tag/tidak menjadi atribut, dimasukkan dalam tag style

  3. eksternal css

    4. berada di luar halaman html/menjadi halaman css baru yang dihubungkan dengan link style css dengan mengetik tag link, atribut rel=stylesheet, dan href=mengarah pada halaman css

  • secara terstruktur, sistem akan mendeteksi style mulai dari inline terlebih dahulu, kemudian internal, dan terakhir eskternal

  • pemanggilan heading 1 - 6 untuk diberi style internal dan eksternal tinggal diketik heading tersebut (misal: h2), maka langsung akan memunculkan format pemberian style nya

  • sedangkan untuk paragraf (p), dapat menggunakan class atau pun id untuk melakukan pemanggilan

  • class dihubungkan dengan mengetik titik (.) terlebih dahulu, misal class="nama" maka diketik .nama, sehingga secara otomatis akan memunculkan format pemberian style

  • sedangkan id dihubungkan dengan mengetik hashtag (#)

  • class bisa digunakan pada beberapa tag, sedangkan id hanya pada 1 tag

Pseudo Class

  • Link:"a:link"

    • digunakan untuk menunjukkan bahwa elemen atau teks yang ditandai sebagai link tersebut akan mengarahkan pada bagian atau halaman tertentu

  • visited:"a:visited"

    • sebagai lanjutan dari link, setelah diklik dan mengarah ke halaman atau bagian lain, maka teks atau elemen tersebut akan berubah warna atau terjadi perubahan yang berbeda dari sebelum di klik

  • hover:"a:hover"

    • digunakan sebagai pertanda ketika kursor diarahkan ke elemen/teks/simbol yang dilabeli hover

  • active:"a:active"

    • berguna ketika kursor diarahkan ke bagian yang dilabeli active, lalu diklik dengan ditahan, kemudian di geser ke arah tertentu, maka akan terjadi perubahan tertentu

Positioning

  • static

    • Posisi tetap/tidak berubah meski diberikan impuls geseran atau lainnya

  • relative

    • Posisi berubah sesuai impuls

  • fixed

    • Posisi elemen akan tetap bahkan jika layar discroll/digeser atas bawah

  • absolute

    • seperti relative, tapi keluar dari bagian yang dilabeli absolute

  • sticky

    • seperti fixed, akan tetapi harus diberikan impuls supaya tetap diposisinya, bergeser jika di scroll

  • z-index

    • menjadi penentu lapisan atas, bawah, tengah, dan urutasn lapisan lainnya
